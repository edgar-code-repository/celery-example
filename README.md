CELERY EXAMPLE APP
---------------------------------------------------------------------------------------------------------------------

Sencilla aplicación desarrollada con Django 2.2 y Celery 4.3 que 
muestra un pequeño ejemplo de ejecución de una tarea asincrónica.

La aplicación utiliza Redis como broker de mensajería.
